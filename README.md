Завдання1. 
Згідно з варіантом обрати рядки для форми та сфортувати її. За допомогою регулярних виразів JS виконати перевірку валідності введеної інформації. 
Якщо все правильно – вивести введену інформацію в окремому вікні, якщо є помилки – виділити рядки, які містять помилку.
